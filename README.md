This is the **complete production-ready website** for **JP Packers & Movers**, a premium moving and relocation company based in Mumbai, India. The site is designed to be modern, fast, mobile-first, and SEO-optimized, scoring above 90 on Google PageSpeed.

## ✨ Features

- **Premium Modern UI** – Glassmorphism, smooth animations, rounded cards, and beautiful gradients.
- **Sticky Header** – With logo, navigation, and a click-to-call button.
- **Smart Quote Form** – Fully functional with **Web3Forms** integration to send leads directly to `jppmofficial@gmail.com`.
- **Google Maps Integration** – Autocomplete for Pickup/Drop addresses, route visualization, and distance calculation (requires API key).
- **Live Review Counter** – Displays **63+ customer reviews** from locations across Mumbai, Thane, and Navi Mumbai for local SEO boost.
- **Gallery with Lightbox** – Hover animations and full-screen image view.
- **15+ FAQs** – Built-in accordion for better user experience.
- **Floating Action Buttons** – Quick access to WhatsApp and Phone calls.
- **Mobile-First Design** – Fully responsive on all devices.
- **SEO Optimized** – Proper meta tags, Open Graph, Twitter Cards, `robots.txt`, `sitemap.xml`, and `manifest.json`.

---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Custom styling with animations, flexbox & grid
- **JavaScript (Vanilla)** – Interactivity, form handling, reviews, gallery
- **Google Maps JavaScript API** – Places Autocomplete & Directions
- **Web3Forms API** – Form submissions without a backend
- **Font Awesome** – Icons
- **Google Fonts** – Poppins typography
